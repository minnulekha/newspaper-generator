## üìñ About

**The Daily Chronicle** is an interactive web tool that allows users to turn their memories into "Breaking News." Whether it's for an anniversary, a birthday, or just capturing a fun moment with friends, this tool lets you design a premium, vintage-themed broadsheet.

Built with vanilla HTML, CSS, and JavaScript, it features a powerful DOM manipulation engine that updates the preview in real-time as you type.

* **üé® 3 Unique Templates:**
    * **The Classic:** A traditional 2-column broadsheet layout.
    * **365 Days Wrapped:** A year-in-review style with bullet points and a footer strip.
    * **The Gallery:** A Polaroid-style snapshot layout for showcasing multiple photos.
* **üñºÔ∏è Advanced Photo Studio:**
    * Upload your own images.

* **üñäÔ∏è Full Customization:**

    * Change the newspaper name and city/location.

## üöÄ How to Use

1.  **Choose a Template:** Use the sidebar to switch between Classic, Wrapped, or Gallery layouts.
2.  **Customize Content:** Type directly into the sidebar inputs to update headlines, stories, and dates.
3.  **Upload Photos:** Select images from your device and choose a filter (B&W/Sepia) to match the vintage aesthetic.
4.  **Add Decorations:** Click to add stamps or tape, then drag them to the perfect spot on the paper.
5.  **Download:** Click the "Download Poster" button to save your creation!
